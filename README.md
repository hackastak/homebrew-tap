# homebrew-tap

Homebrew tap for hackastak tools.

## Usage
```bash
brew tap hackastak/tap
```

Then install any available tool:
```bash
brew install hackastak/tap/repog
```

## Available Tools

| Tool | Description |
|------|-------------|
| [repog](https://github.com/hackastak/repog) | AI-powered knowledge base for your GitHub repositories |
